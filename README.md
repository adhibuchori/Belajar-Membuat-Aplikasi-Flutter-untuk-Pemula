# Shiroko

## 📝 Project Brief
Shiroko is a simple app designed for cat adoption. It showcases a variety of cats available for adoption, providing detailed information about each cat.

## 🛠️ Tech Stack
1. Flutter
2. Visual Studio Code
3. Android Studio
4. Android
5. Figma
